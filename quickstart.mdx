## From Idea to Production

This quickstart guide walks you through transforming your idea into a live web app using Creatr’s AI-powered tools.

### 1. Enter Your Prompt

Start by describing your web app idea in detail. A clear and comprehensive prompt ensures that Creatr generates code that aligns with your vision.

**Example Prompt:**

```plaintext
"Build a modern landing page for a tech startup with interactive UI elements and bold, vibrant colors."
```

### 2. Enhance Your Prompt

Click the **Enhance Prompt** button to let Creatr refine your initial idea. This feature uses advanced AI to add clarity and detail.

- **Review Suggestions:** Consider the AI’s enhancements and adjust as needed.
- **Finalize:** Once satisfied, proceed to submit your prompt.

Hit the **Submit** button to kick off the generation process. Creatr processes your enhanced prompt and produces a tailored codebase for your web app.

### 4. Edit and Customize

Within the Creatr workspace, fine-tune your project:
- **Iterative Edits:** Use Creatr’s AI to make iterative improvements to your code.
- **Component Integration:** Add modular components from leading libraries (e.g., popups, buttons, input fields).
- **OpenAI Integration:** Optionally integrate OpenAI to further enhance your app’s features.

### 5. Export Your Project

Choose one of these export options to save your work:
- **File Download:** Download your project as a ZIP file.
- **GitHub Export:** Automatically create a new GitHub repository with your project code.
- **CLI Export (creatrx):** Clone your project locally using our CLI tool, creatrx.
